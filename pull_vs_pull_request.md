خب پس برای تفاوت پول و پوش و پول ریکوست یه مروری میکنیم : 
پول یعنی : اپدیت کردن 
تغییراتی که روی ریپازیتوری راه دور انجام شده رو میکشی پایین و به نسخه ی خودت اضافه میکنی 
 پوش :  فقط فرستادن تغییراتت به ریپازیتوری خودت 
هیچ‌کس لازم نیست اجازه بده
هیچ‌کس لازم نیست بررسی کنه
فقط می‌فرستی بالا
پول ریکوست : درخواست برای ادغام تغییراتت به ریپازیتوری دیگران (معمولا پروژه ی اصلی ) 
اینجا باید بررسی بشه تایید بشه بعد اگه اوکی بود ادغام میشه
